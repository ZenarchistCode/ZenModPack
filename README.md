# ZenModPack
This is a comprehensive modpack containing all of my stable mods which either don't radically change game mechanics (ie. new items), or can be turned on/off in server-side config.

[Check the Wiki guide here](https://github.com/ZenarchistCode/ZenModPack/wiki) for instructions on how to install and set up all of these mods.

I can't guarantee this modpack will be compatible with other mods - it works with the modlist I'm running at http://HardcoreDayZ.com (Expansion Bundle, Syberia, and a whole bunch of popular mods) but given the complexity of this mod there's guaranteed to be conflicts with some. If you encounter issues, download the source code off my GitHub and modify it/find someone to modify it for you to work around the conflicts.

I actually recommend not repacking this mod if you don't need to - I will be updating this mod regularly. But if you want to you can - just keep in mind any bugfixes and improvements will not be automatically applied, so make sure to check back every few months especially if you encounter any issues/bugs with a particular mod of mine.

As always - the mods in this pack are 100% free, just abide by any copyright rules from 3rd party models etc (I bought a few models off Sketchfab and other sites that require you to credit them if you repack my mod).
